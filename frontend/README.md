# frontend

## Tech Stack

- React-ts (vite)
- shadcn (ui)

## Pre-requisites

- Node v22.19.0

## Structure

```
frontend/
└── src/
    ├── assets
    ├── components/
    │   └── ui
    ├── lib
    ├── services
    ├── types
    ├── App.tsx
    └── main.tsx
```

## Getting Started

```shell
cp .env.example .env
npm install
npm run dev
```
